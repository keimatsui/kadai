* 研究室の美化には，担当者（週替わり）が責任を持つ。
* 充電電池が切れたら自分で充電すること（充電器は赤岡が持っている）。ふつうの乾電池の場合は矢吹のところに持って行って交換すること。
* 退室するときにはPCの電源を切るか，スタンバイあるいは休止状態にすること。
* マウスとキーボードは，マシンに貼ってある番号と同じものを使うこと．交換して使う場合は，関連するすべての責任も負わなければならない。
